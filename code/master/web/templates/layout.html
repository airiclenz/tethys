About Tethys

Tethys is a self-hosted, multi-channel automatic plant-watering system. Battery-powered wireless soil-moisture sensors report to a Raspberry Pi master over a 2.4 GHz nRF24 radio link. The master decides when to water, drives the pumps and valves, and serves this real-time dashboard — all on your own network, with no cloud involved.
